# AdvertisementApp

## 📌 Project Overview

AdvertisementApp is an ASP.NET Core MVC based web application developed for managing advertisements and services.

The project was developed using a layered architecture approach to create a clean, maintainable and scalable application structure.

---

## 🚀 Technologies Used

- ASP.NET Core MVC (.NET 8)
- Entity Framework Core
- SQL Server
- C#
- Repository Pattern
- Unit Of Work Pattern
- AutoMapper
- FluentValidation
- Dependency Injection
- Bootstrap
- JavaScript
- HTML / CSS
